# Frontend Assignment - Kanban Board

A simple Kanban board app using **ReactJS** to fetch and display tasks from an API. The board allows you to group tasks by status, user, or priority and sort them by priority or title. Designed based on the provided screenshots.

## Features
- **Grouping**: Group tasks by Status, User, or Priority.
- **Sorting**: Sort tasks by Priority (desc) or Title (asc).
- **Persistent State**: Keeps your view settings even after page reload.
- **Responsive UI**: Pure CSS, no frameworks used.

## Priority Levels
Tasks are displayed by priority:
- Urgent (4), High (3), Medium (2), Low (1), No priority (0)
